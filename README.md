# 🫙 다썼텅

> 화장품 소비 & 리필 타이밍 메이트

화장품 재고를 관리하고, 텅 버튼 하나로 실제 소진 패턴을 자동 분석하는 PWA 앱이에요.

## 기능

- 📦 화장품 재고 추가 / 수량 관리
- 👆 텅 버튼으로 소진 기록 (재고 1→0 시 파티클 애니메이션)
- 📊 실제 사용 주기 자동 분석 (텅 누를수록 데이터 쌓임)
- 🫙 "일단 쓰고 봄" 모드 — 사용 주기 몰라도 바로 추가 가능
- 📅 세일/구매 일정 연동 및 D-day 관리
- 📱 PWA — 홈 화면에 추가하면 앱처럼 실행

## GitHub Pages로 배포하기

1. 이 레포를 GitHub에 업로드
2. Settings → Pages → Branch: `main` / Folder: `/ (root)` 선택 후 Save
3. `https://{유저명}.github.io/{레포명}/` 으로 접속
4. 브라우저에서 "홈 화면에 추가" 선택 → 앱 설치 완료 📱

## 파일 구조

```
├── index.html     # 메인 앱
├── manifest.json  # PWA 매니페스트
├── sw.js          # 서비스워커 (오프라인 지원)
├── icon-192.png   # 앱 아이콘 (192x192)
└── icon-512.png   # 앱 아이콘 (512x512)
```
